# Evidence Classification Protocol

## Purpose

This protocol defines how evidence is classified in the SAD Moron governance layer.

## Boundary

This protocol governs conceptual evidence meaning only.

It does not define:
- search algorithms
- ingestion routines
- persistence models
- evidentiary automation

## Responsibilities

This protocol is responsible for defining:
- `E+` as explicit evidence of presence
- `E-` as verified absence under declared scope
- the distinction between evidence and accountable decision
- the minimum context required for later reconstruction

## Exclusions

This protocol excludes:
- automatic sanction logic
- legal conclusions by software
- hidden evidence sources
- inferred evidence without declared basis

## Enforcement Rules

### Rule 1: `E+` requires traceable presence

`E+` must refer to evidence that is explicitly present in a declared source or declared review artifact.

### Rule 2: `E-` requires bounded absence

`E-` is valid only when the absence is established under declared source scope, expected event scope, and temporal scope.

### Rule 3: Evidence is not verdict

Neither `E+` nor `E-` should be treated as a final institutional decision by itself.

### Rule 4: Reconstruction readiness

Evidence classification should preserve enough context to allow later review of why the classification was made.

## Summary

Evidence classification in SAD Moron is a governance discipline for presence and bounded absence, not a substitute for accountable institutional judgment.
