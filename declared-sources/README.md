# Declared Sources

This directory registers the operational sources used by SAD Moron workflows after normative authority and workflow scope have been declared.

Use this layer to document:
- Google Sheets
- Drive documents
- Forms
- instructivos

Declared sources support execution and auditability, but they remain operational artifacts and must be distinguished from normative authority documents.

Apps Script design must reference both the governing normative records and the approved workflow records before mapping any declared source.
