# Normative Sources

This directory registers the legal and institutional authority sources that constrain SAD Moron governance and future implementation.

Use this layer to document:
- statute-level sources
- resolutions
- dispositions
- paritarian agreements

This layer defines authority, scope, expected events, rights, obligations, and temporal effects before any operational workflow or Apps Script design proceeds.

Operational sources such as Google Sheets or Drive documents must never be treated as substitutes for normative authority.
